# SQL-Based Analysis of an E-Commerce Sales Dataset

## Project Overview

This project explores an e-commerce sales dataset using Microsoft SQL Server. The analysis was conducted to identify business insights related to customer behaviour, product performance, marketing effectiveness, payment preferences, order fulfilment, and revenue trends.

The project demonstrates the use of SQL for data exploration, aggregation, filtering, grouping, and business intelligence reporting.

## Tools Used

* Microsoft SQL Server
* SQL Server Management Studio (SSMS)
* Microsoft Word
* GitHub

## Analysis Areas

### Dataset Overview

* Total orders
* Unique customers
* Repeat customers

### Product Analysis

* Revenue generated by products
* Units sold by product

### Marketing Analysis

* Referral source performance
* Coupon campaign effectiveness

### Payment Analysis

* Payment method usage
* Revenue contribution by payment method

### Order Status Analysis

* Distribution of order statuses
* Revenue recognition considerations

### Revenue Analysis

* Revenue summary statistics
* Yearly revenue trends
* January–June comparative analysis

### Customer Behaviour Analysis

* Customer activity trends
* Repeat purchasing behaviour

## Key Findings

* FREESHIP generated the highest revenue and average order value among coupon campaigns.
* Instagram produced the highest revenue among referral channels.
* Chairs generated the highest product revenue.
* Credit Card customers recorded the highest average spending.
* Revenue declined gradually between 2023 and 2025.
* Customer numbers, order volumes, and average order values all decreased during the analysis period.
* Repeat purchasing behaviour was extremely limited, indicating a strong reliance on customer acquisition.
